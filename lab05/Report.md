# Лабораторная работа: Моделирование случайных событий

## Цель работы

Разработать приложения для получения случайных ответов с использованием ранее реализованного генератора случайных чисел.

## Приложение 1: "Скажи 'да' или 'нет'"

При нажатии кнопки генерируется случайное число и сравнивается с заданной вероятностью события "Да" (значение от 0 до 1):

- Если случайное число < вероятность — ответ "Да"
- Иначе — ответ "Нет"

**Параметры:** Вероятность "Да" можно менять в диапазоне [0, 1] с шагом 0.01.

## Приложение 2: "Шар предсказаний (Magic 8-Ball)"

Пользователь задаёт вопрос, программа:

1. На основе вероятностей выбирает тип ответа (положительный, нейтральный или отрицательный)
2. Случайно выбирает конкретный ответ из выбранного типа

**Список ответов (18 шт.):**

- Положительные (9 шт.): "Да", "Бесспорно", "Определённо да", "Предрешено" и др.
- Нейтральные (5 шт.): "Спроси позже", "Пока не ясно", "Сконцентрируйся" и др.
- Отрицательные (4 шт.): "Нет", "Весьма сомнительно", "Не рассчитывай" и др.

**Параметры:** Три независимых вероятности (в диапазоне [0, 1]):

- Вероятность положительного ответа (по умолчанию 0.44)
- Вероятность нейтрального ответа (по умолчанию 0.28)
- Вероятность отрицательного ответа (по умолчанию 0.28)

Все вероятности автоматически нормализуются так, чтобы их сумма равнялась 1.

## Реализация

- Генератор случайных чисел (LCG) взят из предыдущей лабораторной работы
- Добавлена возможность изменения зерна для демонстрации воспроизводимости
- Реализована анимация при смене ответов
- **Управление вероятностями:** Оба приложения позволяют менять вероятности в диапазоне [0, 1]
  - Для приложения 1: одна вероятность (события "Да")
  - Для приложения 2: три вероятности с автоматической нормализацией

## Результаты

**Работа приложений:**

- Каждое нажатие даёт новый ответ согласно заданным вероятностям
- При совпадении ответов анимация показывает, что событие произошло
- Изменение зерна позволяет получить другую последовательность ответов
- Динамическое изменение вероятностей позволяет экспериментировать с распределениями:
  - При вероятности "Да" = 0 ответ всегда "Нет"
  - При вероятности "Да" = 1 ответ всегда "Да"
  - При вероятности "Да" = 0.5 ответы распределяются поровну
