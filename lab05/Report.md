# Лабораторная работа: Моделирование случайных событий

## Цель работы

Разработать приложения для получения случайных ответов с использованием ранее реализованного генератора случайных чисел.

## Приложение 1: "Скажи 'да' или 'нет'"

При нажатии кнопки генерируется случайное число. Если меньше 0.5 — ответ "Да", иначе — "Нет".

## Приложение 2: "Шар предсказаний (Magic 8-Ball)"

Пользователь задаёт вопрос, программа случайно выбирает один из 18 ответов:

- Положительные (9 шт.): "Да", "Бесспорно", "Определённо да", "Предрешено" и др.
- Нейтральные (5 шт.): "Спроси позже", "Пока не ясно", "Сконцентрируйся" и др.
- Отрицательные (4 шт.): "Нет", "Весьма сомнительно", "Не рассчитывай" и др.

## Реализация

- Генератор случайных чисел (LCG) взят из предыдущей лабораторной работы
- Добавлена возможность изменения зерна для демонстрации воспроизводимости
- Реализована анимация при смене ответов

## Результаты

**Работа приложений:**

- Каждое нажатие даёт новый ответ
- При совпадении ответов анимация показывает, что событие произошло
- Изменение зерна позволяет получить другую последовательность ответов

## Выводы

1. Собственный генератор случайных чисел успешно применён для моделирования двух типов случайных событий
2. Воспроизводимость генератора позволяет повторять эксперименты
3. Анимация делает интерфейс наглядным
4. Приложения демонстрируют базовые принципы стохастического моделирования
