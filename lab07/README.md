# Марковская модель погоды

**Задание:**  
Смоделировать погоду по дням:

- 1 — ясно  
- 2 — облачно  
- 3 — пасмурно  

Единица времени — **1 день**.  
Задать интенсивности переходов между состояниями.

**Требования:**

1. Выполнить моделирование в «реальном» времени с визуализацией.
2. Провести статистическую обработку результатов.
3. Сравнить эмпирическое распределение с теоретическим стационарным.
4. Сбор статистики и всей необходимой информации в .txt или .csv формат (.csv формат предпочтительней)

---

## Отчёт о выполненной работе

В рамках данной лабораторной работы была разработана программа для имитационного моделирования смены погоды с использованием аппарата цепей Маркова с непрерывным временем.

**Реализованный функционал:**

1. **Построение модели:**
   - Состояния системы: Ясно, Облачно, Пасмурно.
   - Пользовательский интерфейс позволяет задавать внедиагональные элементы матрицы интенсивностей переходов ($Q$). Параметры ухода из состояния ($\lambda_i$) вычисляются на основе введенных переходов.

2. **Проведение симуляции:**
   - **Мгновенно:** Моделирование выполняется до заданного момента времени, после чего выводятся итоговые результаты.
   - **В «реальном» времени (Анимация):** 1 день модельного времени пролетает за ~1 секунду реального времени. Визуально отображается смена состояний и отсчитывается прошедшее время (в днях и часах).
   - Присутствует кнопка **Сброс** для корректной остановки анимации, очистки статистики и возврата к стартовому состоянию.

3. **Статистическая обработка:**
   - **Теоретические вероятности:** Рассчитываются аналитически путем решения системы линейных уравнений $\Pi \cdot Q = 0$ и $\sum \pi_i = 1$ методом Крамера.
   - **Эмпирические вероятности:** Оцениваются как доля времени нахождения цепи в определенном состоянии ($T_i / T_{max}$) за всё время моделирования.
   - Выводится таблица с абсолютными ошибками отклонения эмпирической оценки от теоретической.

4. **Визуализация и экспорт данных:**
   - Для наглядного сравнения эмпирических и теоретических распределений используется столбчатая диаграмма на базе библиотеки **Chart.js**.
   - Детальный лог смены состояний и продолжительности пребывания в каждом из них сохраняется и доступен для выгрузки в формате **.csv**.
